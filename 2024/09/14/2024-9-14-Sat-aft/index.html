1.如何评价计算机设计的好坏

  • 响应时间/执行时间Response time/execution time
    • 运行时间Elapsed time
    • CPU时间CPU time
  • 吞吐率Throughput
    • 单位时间内能做的事

时钟周期Clock period

  • 以上沿/下沿为起点计算
  • 时钟频率 = 1s / 时钟周期
  • 1 ps = 10^-12 s
  • 1GHz = 10^9 Hz
    • CPU时间 = CPU时钟周期数 * 时钟周期 = CPU时钟周期数 / 时钟频率
    • 执行一条命令的时钟周期数CPI = 总时间周期数 / IC
      • 因此可以通过降低时钟周期数 增加时钟频率 平衡时间和周期数实现增效
  • 指令集总命令数IC,Instruction Count
    • 取决于程序，指令集架构ISA和编译器compiler

计算机的表现取决于：

  • 算法：影响IC，有可能影响CPI
  • 编程语言：影响IC,CPI
  • 编译器：影响IC,CPI
  • 指令集架构Instruction set architecture,ISA：影响IC,CPA,时钟周期
