计算机组成Outline

what is a computer?

计算机发展历史

Pascaline:十七世纪出现的计数器：

可以直接实现加减，间接实现乘除

  • 图灵机
  • 冯诺依曼计算机
    • 计算与存储分离
    • 数据与指令存在同一个存储器
  • 第二代计算机
    • 时期：1956-1964
    • 特征：晶体管代替电子管
    • 编程语言从二进制01进入符号语言
  • 第三代计算机
    • 时期：1964-1971
    • 特点：集成电路Intergranted Circulate代替晶体管
    • 操作系统允许同时运行多个应用程序
    • 商业上极为成功的代表：IBM/360 family
    • 操作系统出现了虚拟内存和分时的概念
  • 第四代计算机
    • 时期：1971-现在
    • 更小，更快，更便宜
    • 特征：微处理器Microprocessor代替集成电路

RISC Architecture精简指令集计算机架构与当代计算机

RISC(Reduced Instruction Set Computer) 

指令执行用尽量少的时钟周期
指令编码定长
->带来优势：CPU与编译效率高  
->->在ARM芯片，智能手机上广泛使用

So what is a computer?

计算机是一个系统

硬件划分

interface接口
datapath数据通路
control unit控制单元

软件划分（按用途分类）

Operation system操作系统
Firmware驱动：专为一种硬件设计的软件

memory存储

Main memory主存

  • 易失性
  • 用于在程序运行时维持其运行
  • 例子：计算机上的
    • DRAM动态随机存储器Dynamic Random Access Memory
    • SRAM静态随机存储器Static Random Access Memory

second memory外存储器

  • 非易失性
  • 用于在程序运行的间隔存储程序与数据
  • 例子：
    • 固态硬盘Hard disk，PMD的闪存Flash Memory

易失性Volatile

是指存储器断电后，它存储的数据内容是否会丢失的特性。一般易失性存储器存取速度快，而非易失性存储器可长期保存数据。

High-level programming language高级语言：A+B（编程语言，c，python）
