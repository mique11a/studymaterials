1.样本空间与随机事件

世界上存在两种现象：

  • 确定性现象：结果确定
  • 不确定现象：结果不确定

概统是研究随机变量与数量规律的学科

以抛硬币为例，nH表示正面Head向上的次数，fnH表示正面向上的频率
据图可知，不确定现象在个别实验中结果呈现不确定性，但在大量重复试验中其结果又具有统计规律

随机试验与随机事件

对随机现象的观察记录实验成为随机试验，具有以下特性：

  • 可在相同条件下重复试验
  • 事先知道所有可能的结果
  • 试验前不知道发生哪一个结果

随机试验E的所有可能结果构成的集合称为E的样本空间，记为S，S中的元素e为样本点

  • 例如，抛硬币E的样本空间S = {正面，反面}，掷骰子的样本空间S = {1,2,3,4,5,6}
  • 样本空间可以是有限的掷硬币，也可以是无限的不考虑其他条件下的上车人数；可以是可数的上车人数，也可以是不可数的灯泡寿命

随机事件：S的子集A为E的随机事件A，简称事件A

  • 当且仅当A包含的一个样本点发生称为事件A发生
  • 一个样本点组成的单点集称为基本事件
  • 每次事件S全集必然发生，称S为必然事件
  • 每次空集不可能发生，称为不可能事件
  • 事件有包含，相等两种关系，和集合中的定义相同
    • 若A包含于B，那么A发生，B必定发生
    • 若AB相等，那么A包含于B且B包含于A

事件运算：

  • A与B的和事件,记为A ∪ B
    • 等同集合运算中的并
    • 表示A发生或B发生
  • A与B的积事件，记为A ∩ B
    • 等同集合运算中的交
    • 表示A,B同时发生
  • A ∩ B = 空集,表示AB互不相容或互斥
    • 表示AB不会同时发生
  • A = B的补集，称为A,B互逆，AB互为对立事件
    • 表示A发生B一定不发生，A不发生B一定发生

熟记达摩根定律！

2.频率与概率

频率用来衡量事件发生的频繁程度

  • 不可能事件的频率是0.必然事件的频率是1

频率的性质：

  • 0 <= f <= 1
  • 若事件A1A2A3……两两互斥，那么A1 ∩ A2 ∩ A3……的频率是A1A2A3各自频率之和
    • 这一性质称为可列可加性

当尝试次数趋于无穷大时，事件A的频率收敛于一个稳定的值P(A)，称为事件A的概率

  • 据此，概率具有和频率相同的性质

正式的定义：

  • 对S中的事件A定义一个数P(A)，满足：
    • 非负性： P(A) >= 0
    • 规范性： P(S) = 1
    • 可列可加性：同上
  • 则称P(A)为A的概率

概率的性质：

  • P(∅) = 0
  • A的对立事件发生的概率 = 1 - P(A)
  • P(B-A) = P(B) - P(B ∩ A) = P(B ∪ A) - P(A)
  • P(A ∪ B) = P(A) + (B) - P(A ∩ B)
    • 推广：三个事件的并的公式
    • 推广：n个事件的并的公式

3.等可能概型(古典概型)

若试验E满足：

  • 有限性：S中样本点有限
  • 等可能性：出现每一种结果的可能性相同

那么称这种模型是等可能概型

tips

高中很经典的摸球等等模型都是古典概型，特点是球的大小相等，摸到每个球的概率是相等的，不是每种颜色结果的的概率是相等的！所以它是古典概型

一个课外问题：生日不是随机分布的，部分月份出生的孩子明显多于其他月份，所以这应该不是古典概型？

生日问题求解

全班n个同学，第一个同学有365种选择，第二个为了不重复有364种……总共的情况有365^n个情况，不重复的有365*364 * …… * (365-(n-1))种，除一下就能得到结果了

抽签问题求解

问题：

设抽取的顺序为一个排列，那么总共的排列有n！种，限定第k位是红球，相当于k位上a种取红球的方法 * 剩下来的n-1个球全排列，于是概率为：

P(A) = a * (n-1)! / n! = a/n

或将第k次取到的球号作为样本点，可以视为等概率事件，每个球抽到的概率相同，因此概率仍为a/n
